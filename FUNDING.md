# Funding

This website is part of my ongoing work on Open Cloud Mesh (OCM). It is where I
try to give OCM a clear public home and, through the Observatory, an honest and
browsable view of how implementations actually interoperate. Keeping that up,
with real evidence behind every result, takes sustained time, and that time is
possible because someone chose to fund open source infrastructure directly.

Work on this repository is supported by the Sovereign Tech Agency through the
Tech Fund, under the Open Cloud Mesh project. I am grateful for that support.
